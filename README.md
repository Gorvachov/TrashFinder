¿En qué consiste el proyecto?
Nuestro proyecto lleva por nombre Trash Finder, y se trata de una aplicación pensada tanto para los ciudadanos como para los recolectores de la ciudad de Lima. Para los ciudadanos, les ayudará a localizar contenedores inteligentes en tiempo real para que puedan desechar sus residuos, y se les otorgará puntos por sus buenas prácticas. En el caso de los recolectores, la aplicación les agilizará su proceso de recolección de basura indicándoles la ruta más óptima para su jornada.
